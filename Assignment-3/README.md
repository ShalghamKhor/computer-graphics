# Assignment 3

This assignment builds a Python/Pygame OpenGL solar-system simulation with shader-based lighting, animated planet/moon orbits and rotations, plus a separate data-point visualization using custom point shaders.

### Requirements
- Python 3.10+
- `pygame`
- `PyOpenGL`
- `PyGLM`
- `numpy`

Install dependencies:
```bash
python3 -m pip install pygame PyOpenGL PyGLM numpy
```

### Run Solar System Demo
```bash
python3 solar_main.py
```
